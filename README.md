# Calorie Tracker Web App

A simple, modern web application to track your daily calorie intake.

## Features

- ✅ Add food items with calories
- ✅ Set custom daily calorie goals
- ✅ Visual progress bar showing consumption
- ✅ Delete individual items or clear all
- ✅ Data persists using local storage
- ✅ Responsive design for mobile and desktop
- ✅ Clean, modern UI

## How to Use

### Starting the Server

**Option 1: Using the batch file (Windows)**
- Double-click `start-server.bat`
- The server will start and your browser will open automatically

**Option 2: Using npm**
- Run `npm start` in the terminal
- The server will start on http://localhost:8000

**Option 3: Direct file access**
- Simply open `index.html` in your web browser

### Using the App

1. **Set your goal**: Enter your daily calorie goal and click "Set Goal"
2. **Add foods**: Enter food name and calories, then click "Add"
3. **Track progress**: Watch the progress bar fill as you add items
4. **Manage items**: Delete individual items or clear all at once

- Chrome
- Firefox

## Installation

No installation required! Just download the files.

### Quick Start

1. **Windows**: Double-click `start-server.bat`
2. **Command Line**: Run `npm start`
3. **Direct**: Open `index.html` in your browser

The app will be available at http://localhost:8000

## Data Persistence

All data is stored locally in your browser using Local Storage. Your calorie goal and food items will persist even after closing the browser.
